# App Icon Assets

Place the following files in this directory:
- `app_icon.png` — 1024x1024 app icon (full icon for iOS)
- `app_icon_foreground.png` — 1024x1024 adaptive icon foreground (Android)

## Generate icons

After placing your icon files, run:
```bash
dart run flutter_launcher_icons
```

## Suggested design
- Background color: #1A1A2E (dark navy)
- Foreground: headphones icon with playlist waves, white/gradient
- Style: minimal, rounded corners
