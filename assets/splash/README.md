# Splash Screen Assets

Place the following file in this directory:
- `splash_logo.png` — 288x288 centered logo for splash screen

## Generate splash

After placing your logo file, run:
```bash
dart run flutter_native_splash:create
```

## Suggested design
- Logo: same headphones icon from app icon, white on transparent
- Background: #1A1A2E (matches app theme)
