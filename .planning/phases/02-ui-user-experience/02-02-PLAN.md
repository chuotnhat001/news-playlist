Create lib/features/player/mini_player.dart:
- ConsumerWidget that watches audioPlayerProvider.
- Renders nothing (SizedBox.shrink) when playlist is empty.
- When playing or paused: a Container with height ~64, Material 3 surface color, slight elevation.
- Row contents: leading Icon (Icons.music_note), Expanded Column with title (currentArticle.title, maxLines 1, overflow ellipsis) and subtitle (currentArticle.source), IconButton play/pause toggle (calls resume/pause on notifier), small LinearProgressIndicator at bottom showing position/duration ratio.
- Wrap entire widget in GestureDetector — onTap navigates to PlayerScreen (Navigator.push).

Create lib/features/player/widgets/progress_bar.dart:
- ConsumerWidget showing a Slider widget.
- Reads position and duration from audioPlayerProvider.
- Slider value = position.inMilliseconds, max = duration.inMilliseconds (handle zero duration).
- onChanged calls ref.read(audioPlayerProvider.notifier).seekTo(Duration(milliseconds: value)).
- Row below slider with position label (mm:ss) on left, duration label on right.
- Helper function to format Duration as mm:ss string.

Create lib/app_shell.dart:
- ConsumerWidget that wraps child content with the MiniPlayer overlaid at the bottom.
- Use a Column: Expanded(child: child), MiniPlayer().
- This widget will be used in main.dart to wrap the navigator/home screen so mini player persists across navigation.
