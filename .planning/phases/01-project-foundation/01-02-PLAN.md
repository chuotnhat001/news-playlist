  <behavior>
  - insertArticles(List of Article) stores articles in SQLite using INSERT OR REPLACE
  - getArticlesByCategory(category) returns articles WHERE category matches AND cachedAt within 6h, ordered by publishedAt DESC
  - isStale(category) returns true if no articles in category OR oldest cachedAt exceeds 6 hours
  - clearExpired() deletes rows WHERE cachedAt less than (now - 6 hours)
  - clearAll() deletes all rows from articles table
  - init() opens database and creates articles table if not exists using Article.createTableSQL
  - DateTime stored as millisecondsSinceEpoch (int) in SQLite
  </behavior>
  <action>
  Create lib/services/cache_service.dart:

  Class CacheService with:
  - Static const cacheTTL = Duration(hours: 6)
  - Late Database _db field
  - init() method: use path_provider getApplicationDocumentsDirectory to build DB path, call openDatabase with Article.createTableSQL in onCreate
  - initWithDatabase(Database db) method: for testing with in-memory DB
  - insertArticles(List of Article): batch insert using transaction, INSERT OR REPLACE conflict algorithm. Use article.toMap() for values.
  - getArticlesByCategory(String category): query WHERE category = ? AND cachedAt > (now - 6h in ms), ORDER BY publishedAt DESC. Map results via Article.fromMap.
  - isStale(String category): query COUNT WHERE category = ? AND cachedAt > cutoff. Return true if count == 0.
  - clearExpired(): DELETE WHERE cachedAt < cutoff milliseconds
  - clearAll(): DELETE from articles table (no WHERE)
  - Private _cutoffMillis() helper: DateTime.now().subtract(cacheTTL).millisecondsSinceEpoch

  Import article.dart model. Import sqflite and path_provider.
