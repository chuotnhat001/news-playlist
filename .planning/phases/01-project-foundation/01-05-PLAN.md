<task type="auto" tdd="true">
  <name>Task 2: AudioPlayerProvider — Riverpod state with playlist, auto-next, skip-on-error</name>
  <files>lib/providers/audio_player_provider.dart, test/providers/audio_player_provider_test.dart</files>
  <behavior>
  - setPlaylist(articles) stores playlist, sets currentIndex=0, starts playing first article
  - play() starts playback of current article
  - pause() pauses, resume() resumes
  - skipNext() increments currentIndex and plays next article. At end of playlist: stops playback.
  - skipPrevious() decrements currentIndex and plays previous article. At index 0: restarts current.
  - seekTo(position) delegates to audio service seek
  - On track completion (playerState==completed): automatically calls skipNext (auto-next)
  - On playback error: automatically calls skipNext (skip-on-error)
  - State exposes: currentArticle, playlist, currentIndex, isPlaying, position, duration, isLoading, error
  - Empty playlist: all playback methods are no-ops
  </behavior>
  <action>
  Create lib/providers/audio_player_provider.dart:

  Define immutable AudioPlayerState class:
  - List of Article playlist (default empty)
  - int currentIndex (default 0)
  - Article? get currentArticle => playlist.isEmpty ? null : playlist[currentIndex]
  - bool isPlaying (default false)
  - Duration position (default Duration.zero)
  - Duration duration (default Duration.zero)
  - bool isLoading (default false)
  - String? error (default null)
  - copyWith() method

  Define AudioPlayerNotifier extends StateNotifier of AudioPlayerState:
  - Constructor takes NewsAudioService instance
  - In constructor: listen to audioService.playerStateStream:
    - On completed: call _onTrackComplete()
    - On error: call _onPlaybackError()
  - Listen to audioService.positionStream: update state.position
  - Listen to audioService.durationStream: update state.duration

  - void setPlaylist(List of Article articles):
    state = state.copyWith(playlist: articles, currentIndex: 0, error: null)
    _playCurrentTrack()

  - void play(): _playCurrentTrack()
  - void pause(): audioService.pause(), state = state.copyWith(isPlaying: false)
  - void resume(): audioService.resume(), state = state.copyWith(isPlaying: true)

  - void skipNext():
    if currentIndex >= playlist.length - 1: stop playback, state isPlaying=false
    else: state = state.copyWith(currentIndex: currentIndex + 1, error: null), _playCurrentTrack()

  - void skipPrevious():
    if currentIndex <= 0: seekTo(Duration.zero) (restart current)
    else: state = state.copyWith(currentIndex: currentIndex - 1), _playCurrentTrack()

  - void seekTo(Duration position): audioService.seek(position)

  - void _playCurrentTrack():
    if playlist.isEmpty return
    state = state.copyWith(isLoading: true, error: null)
    audioService.playUrl(currentArticle!.audioUrl)
    audioService.setMediaItem(currentArticle!.title, null)
    state = state.copyWith(isPlaying: true, isLoading: false)

  - void _onTrackComplete(): skipNext()  // R4: auto-next
  - void _onPlaybackError(): state = state.copyWith(error: 'Playback failed'), skipNext()  // R4: skip on error

  - Wire audioService.onSkipNext = skipNext
  - Wire audioService.onSkipPrevious = skipPrevious

  Define providers:
  - final audioServiceProvider = Provider of NewsAudioService ((ref) => NewsAudioService())
  - final audioPlayerProvider = StateNotifierProvider of AudioPlayerNotifier, AudioPlayerState ((ref) => AudioPlayerNotifier(ref.read(audioServiceProvider)))

  - Test: pause → isPlaying false

  - Test: multiple errors in sequence → skips through playlist until end
